Foi concluido o desafio do Felipão de Classificador de Nível de Herói por XP, para resolver eu criei 2 funções, sendo a primeira para checar o nivel do heroi conforme foi pedido, e a segunda função foi para que toda vez que atualizasse a xp do herói, ele executava o texto pedido atribuindo um novo nível.
--------------------------------------------------------------------------------------------------------------------------------
Objetivo:
Aplicar conceitos básicos de JavaScript, como:
- Variáveis
- Funções
- Condicionais
- Operadores matemáticos

Tecnologias utilizadas:
- JavaScript
- Node.js
- Git e GitHub

Lógica do projeto:
- O herói possui um nome e um valor de XP inicial
- A função `checagemNivel` determina o nível do herói com base no XP
- A função `atualizarHeroi` altera o XP do herói e exibe o nível correspondente

Classificação de níveis:
- Até 1000 XP: Ferro  
- 1001 a 2000 XP: Bronze  
- 2001 a 5000 XP: Prata  
- 5001 a 7000 XP: Ouro  
- 7001 a 8000 XP: Platina  
- 8001 a 9000 XP: Ascendente  
- 9001 a 10000 XP: Imortal  
- Acima de 10000 XP: Radiante  

Exemplo de saída:
- O Herói de nome Toretto está no nível Ferro
